## Radar App 

Radar Application, which is responsive and compatible with every screen, where you can see flight details, status and route, was created using React libraries.


## Kütüphaneler
- @reduxjs/toolkit
- react-redux
- axios
- bootstrap
- leaflet
- react-leaflet
- react-paginate
